# Fairway Forward — smart-coach
**Bishop Chatard HS Boys Golf · D1-Style Coaching Platform**

---

## What This Is

A coaching platform for Bishop Chatard Boys Golf. Swing notes, season scores, events, carpool, tryouts, player profiles, and a public feed — built for coaches, players, and parents.

## Stack

- **Frontend:** Static HTML (`public/index.html`) served from Vercel
- **Backend:** Vercel serverless API routes (Sprint 2)
- **Database:** Supabase Postgres (Sprint 2)
- **Auth:** Custom PIN system via Twilio SMS (Sprint 2)
- **AI:** Anthropic Claude (swing note generation)
